# AlgoViz

> Interactive platform for visualizing data structures and algorithms.

## Overview

AlgoViz is a React-based web application that helps users understand algorithms through interactive visualizations, animations, and execution controls.

---

## Features

- Sorting Algorithm Visualizations
- Graph Algorithm Visualizations
- Searching Algorithms
- Adjustable Animation Speed
- Step-by-Step Execution
- Complexity Analysis
- Responsive UI

---

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

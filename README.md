# AlgoViz

Interactive Algorithm Visualizer built with React, TypeScript and Tailwind CSS.

## Features

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### Searching
- Linear Search
- Binary Search

### Graph Traversal
- Breadth First Search (BFS)
- Depth First Search (DFS)

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
